# Python Games

This project uses [Vite](https://vitejs.dev/) with React to power local development and builds.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Starts a development server on [http://localhost:5173](http://localhost:5173) with hot module replacement.

### `npm test`

Runs the test suite using [Vitest](https://vitest.dev/).

### `npm run build`

Builds the app for production in the `docs` directory so that it can be published as a static site (for example, with GitHub Pages).

### `npm run preview`

Serves the production build locally for verification.
